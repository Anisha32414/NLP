# Natural Language Processing (NLP)

A collection of Natural Language Processing (NLP) concepts and implementations, covering text preprocessing, tokenization techniques, feature engineering, word embeddings, and deep learning architectures for sequence modeling.

## Topics Covered

### Text Preprocessing
- Text Cleaning
- Stopword Removal
- Tokenization
  - `sent_tokenize`
  - `word_tokenize`
- Stemming
  - PorterStemmer
  - RegexpStemmer
  - SnowballStemmer
- Lemmatization

### Types of Encoding
- One Hot Encoding
- Bag of Words (BoW)
- TF-IDF

### Word Embedding
- Word2Vec
  - CBOW (Continuous Bag of Words)
  - Skip-Gram
- AvgWord2Vec

> **Practical Implementation:** Word2Vec implementation is available in **`Word2Vec-Implementation.ipynb`**.

### Deep Learning for NLP
- Recurrent Neural Networks (RNN) 
- Long Short-Term Memory (LSTM RNN) 
- Bidirectional LSTM

## Tech Stack
- Python
- NumPy
- Pandas
- NLTK
- Gensim
- Scikit-learn
