# Natural Language Processing (NLP)

A collection of Natural Language Processing (NLP) concepts and implementations, covering text preprocessing, tokenization techniques, feature engineering, and deep learning architectures for sequence modeling.

## Topics Covered
- Text Preprocessing
- Tokenization
- Stopword Removal
- Stemming
- Lemmatization
- Text Cleaning
- Bag of Words (BoW)
- TF-IDF
- Word Embeddings
- Word2Vec
- AvgWord2Vec
- Recurrent Neural Networks (RNN)
- Long Short-Term Memory (LSTM)
- Gated Recurrent Unit (GRU)

## Tech Stack
- Python
- NumPy
- Pandas
- NLTK
- Scikit-learn
- TensorFlow / Keras
